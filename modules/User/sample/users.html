? User File Format ตัวอย่างการสร้างไฟล์

เราสามารถเริ่มสร้างข้อมูลผู้ใช้งานจากไฟล์ฐานข้อมูล เช่น Access หรือจาก excel แล้ว export มาอยู่ในรูปของ text file ขั้นด้วยตัวอักษรพิเศษ จากนั้นนำไป import เข้าระบบได้เลย

ข้อควรระวัง
ไฟล์ที่ใช้ต้องทำการEncodeแบบ UTF-8 เท่านั้น ถ้ามีปัญหาลอง Convert ไฟล์ให้เป็น UTF-8 without BOM ด้วยโปรแกรมต่างๆเช่น
Dreamweaver
NotePad++
Open Office

ตัวอย่างเช่น

1. เริ่มสร้างจากไฟล์ excel ซึ่งประกอบด้วยข้อมูลนักเรียนชั้น ม. 1 โดยข้อมูลบรรทัดแรกใช้อ้างถึงชนิดช้อมูลที่จะส่งให้ระบบ เช่น ln_uno(เลขประจำตัว) , ln_name(ชื่อนามสุกลจริง), ln_uname(ชื่อที่ใช้ login), ln_pass(รหัสผ่าน), ln_email(อีเมล์),ln_phone(โทรศัพท์),ln_news(รับข่าวสาร)



2. เลือกเมนู แฟ้ม(File) -> บันทึกเป็น (Save as) -> เก็บเป็นชนิด (Type)
ตั้งชื่อ lnusers.csv



3. จะได้ text file ที่มีหน้าตาดังนี้ซึ่งจะเป็นข้อมูลคั่นด้วย , (comma)
